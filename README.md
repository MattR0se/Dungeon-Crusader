# Dungeon-Crusader

Start main.py to play

### Controls:
| key | action|
|-----|-------|
| ARROW KEYS or WSAD | move the player|
| SPACE | Sword attack |
| ESC | open inventory |
| F6 | quicksave (might crash the game!)|
| F9| quickload |
| H | debug mode |

### Controls in debug mode:
| key | action|
|-----|-------|
|F4 | slow motion mode (5 FPS)|
|PAGEUP/PAGEDOWN | change hp/mana|
